# LG gram Pro 16

## 位置づけ

- 暫定的な購入候補は `16Z90U-KUB5J`
- 16インチ帯で画面サイズと持ち運びやすさの両立を狙う本命候補
- LG 公式ページで主要スペックを確認済み

## 注目理由

- `Core Ultra X7 358H / 32GB` で、現行機より CPU 性能とメモリ余裕に期待できる
- `16型 2880 x 1800 OLED` により、現行端末より広い作業領域を確保しやすい
- `1,199g` とかなり軽く、持ち運び条件への一致度が高い

## 公式ページで確認したスペック

| 項目 | 内容 |

| 型番 | `16Z90U-KUB5J` |
| シリーズ | `LG gram Pro 16` |
| CPU | `Intel Core Ultra X7 358H` |
| メモリ | `32GB LPDDR5X` |
| ストレージ | `512GB SSD` |

| 解像度 | `2880 x 1800` |

## 開発用途で確認したい点

- Docker や WSL を併用したときの体感速度
- Flutter ビルド時間と熱の上がり方
- 16インチで日常開発に十分な一覧性があるか
- 17インチモデルとの冷却差や継続性能差がどの程度あるか
- `512GB` をそのまま許容するか、上位容量構成を待つか

## メリット

- `32GB` と `Panther Lake 世代` の条件を満たしている
- `1,199g` で重量条件にかなり強い
- 日本向け公式ページで `日本語キーボード` を確認できる
- 現行端末より作業領域を広げやすい

## 懸念点

- 17インチと比較して画面サイズ差が小さくないか
- 価格差に対して、満足度の差が説明できるか
- 軽量化の代償として剛性や冷却に妥協がないか
- `512GB SSD` が開発用として足りるかは使い方次第
- 発売直後の実機レビューがまだ少ない可能性がある

## 暫定結論

- 現時点では `16Z90U-KUB5J` を第一候補として扱う
- 理由は `32GB / 日本語キーボード / 1,199g / 高解像度 16型 OLED` が一次選定基準にかなり合うため
- 今後は価格、実機レビュー、17インチモデルとの比較で最終判断する
