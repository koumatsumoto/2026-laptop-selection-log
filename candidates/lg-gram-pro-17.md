# LG gram Pro 17

## 位置づけ

- 現時点の最有力候補
- 17インチ級の広い画面と軽量性の両立を期待している

## 注目理由

- 画面が広く、VS Code とブラウザとターミナルを並べやすい
- 現行端末の「画面がやや小さい」という不満を解消しやすい
- 17インチ帯で軽ければ、持ち運び負担も下げられる可能性がある

## 開発用途で確認したい点

- Docker のイメージビルドやコンテナ起動が十分に速いか
- Flutter のビルド時間が現行端末よりどの程度改善するか
- WSL 利用時にファイル I/O やメモリ使用量で不満が出ないか
- 高負荷を継続したときに熱で性能が落ちすぎないか

## メリット

- 画面が広く開発作業の一覧性を上げやすい
- 現行端末の画面サイズへの不満を解消しやすい
- 17インチ帯で軽量なら携帯性との両立が見込める

## 懸念点

- 薄型軽量設計と継続高負荷の相性
- 大画面モデルとしての実質携帯性
- 構成によっては価格が上がりやすい可能性
