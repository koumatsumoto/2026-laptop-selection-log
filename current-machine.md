# 現行機メモ

## 概要

- 機種: `Surface Laptop 4`
- サイズ: `15インチ`
- 型番: `1979`
- 用途: 比較の基準機

## 現在の使用感

- TypeScript / Python 開発、VS Code、AI を使ったコーディングでは大きな不満はない
- ブラウザ利用も不満はない
- Docker の起動や通常利用は問題ない
- Docker ビルドは遅い
- Flutter ビルドは遅いが、定常的には使っていない
- ストレージが遅くなってきているように感じる
- 端末がやや重い
- 15インチでも画面がやや小さいと感じる
- タッチディスプレイは全く使っていない
- タッチによる誤操作がある
- タッチ対応はバッテリー持ちにも不利だと感じており、次は不要

## 画面の基準

- 解像度: `2496 x 1664`
- 表示スケール: `150%`
- 実効解像度の目安: `約 1664 x 1109`
- 文字の読みやすさは快適
- 次期候補は物理解像度だけでなく、拡大後の実効作業領域で比較する

## バッテリーの基準

- 現状は `4時間作業で 0 になる程度`
- 特に大きく困ってはいない
- 次期候補では `外出時に 10時間持てば十分`
